# Zenithx

**Zenithx** is a custom-built, responsive website for a fictional AI-powered SaaS platform. Created as a portfolio project to showcase web development skills, it features modern UI/UX, dashboard mockups, and clean, maintainable code. All content is placeholder — no real functionality — demonstrating expertise in HTML, CSS, and JavaScript. 

---

## Features

- **Responsive Design:** Fully mobile friendly layout that adapts seamlessly across devices.  
- **Modern UI/UX:** Clean, professional SaaS-style interface with smooth navigation and clear visual hierarchy..
- **Interactive Elements:** Hover effects, buttons, accordions (FAQ), and call-to-action sections.  
- **Dashboard Mockups:** Realistic analytics panels with charts, metrics, and agent status placeholders.  
- **Custom Styling:** Precise typography, color palette, gradients, and subtle animations.  
- **Semantic HTML:** Clean, accessible markup for better SEO and readability.  
- **Component-Based Structure:** Reusable sections (hero, features, testimonials, footer) for easy maintenance.  
- **Performance Optimized:** Lightweight assets and efficient code structure.  
- **Placeholder Content Integration:** Professional use of mock logos, testimonials, and data visualizations.
